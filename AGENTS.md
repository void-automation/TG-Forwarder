# AGENTS.md

Instructions for contributors/agents working in this repository.

## Scope
These instructions apply to the whole repository.

## Development
- Keep the script user-account based (Telethon user sessions), not bot-token based.
- Prefer environment-variable configuration over hardcoded values.
- Keep dependencies minimal.

## Validation
- Run at least a syntax check before committing.
- Update docs when behavior/configuration changes.
